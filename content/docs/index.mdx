Arkitekt is an open-source platform for bioimage analysis — a "middleman" that
sits between your data, your tools and your team. These docs cover installing the
platform, understanding how it works, and building your own apps.

## Where to start

- **[Get started](/docs/introduction/installation)** — Install the platform and
  connect your first instance with the Installation Guide.
- **[Technical design](/docs/design/why)** — Curious how it fits together? The
  Technical Explanation covers what the middleman does and why.
- **[Tutorials](/docs/introduction/first_steps)** — Spend your first few minutes
  with Arkitekt: data management, analysis execution and dashboarding.
- **[Developer guide](/docs/developers)** — Go deeper and start building bioimage
  apps on top of the platform.

## Community & support

Arkitekt is developed in the open. Get help, follow development, and contribute
