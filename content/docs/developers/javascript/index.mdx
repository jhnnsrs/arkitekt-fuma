## Typescript/Javascript

Arkitekt aims to provide a seamless experience for both Typescript and Javascript users, that want to interact with the 
Arkitekt Server in a typesafe manner.

### Scope of this documentation

This documentation will cover the following topics:

- [Installation](/docs/developers/javascript/installation)
- [Usage](/docs/developers/javascript/usage)

<Callout type="info">


This documentation is still a work in progress, and will be updated as we go along. Currently we focus on the ability to
interact with the Arkitekt Server **without** providing functionality to it. While it is perfectly possible to do so, we
will not cover it in this documentation at this point. Deadlines...
