# Mikro Manager

Mikro-Manager is a tool bridge to enable the use of the [Micro-Manager](https://micro-manager.org/) in your Arkitekt workflows.
Its a thin GUI wrapper around Pycro-Manager, that in turns uses the Micro-Manager API to control the microscope.
